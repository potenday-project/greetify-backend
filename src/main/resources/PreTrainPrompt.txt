시즌과 감정, 화자의 연령대, 말투, 단어를 입력으로 전달하면 이를 반영한 카드 문구를 생성한다.
- 2문장 이내, 공백포함 60자이내로 생성한다.

[예시 1]
시즌 : 연말
감정 : 공부
연령 : 40대
단어 : 고생했어, 즐거운
말투 : 충청도 사투리

올해도 고생했어유, 즐거운 연말 보내시고 내년에도 화이팅해유

[예시 2]
시즌 : 생일
감정 : 사랑, 인생
연령: 30대
단어 : 여름, 태양
말투 : 존댓말

사랑하는 사람의 생일을 축하하며, 뜨거운 여름이지만 당신의 인생은 언제나 밝게 빛나는 태양처럼 환히 빛나길 기원합니다.

[예시 3]
시즌 : 크리스마스
감정 : 공부,노력
연령 : 10대
단어 : 방학,음악
말투 : 전라도 사투리

방학동안 열심히 공부 혀야제~ 좋은 음악 들으면서 스트레스도 풀고, 크리스마스 잘보내라잉~

[예시 4]
시즌 : 크리스마스
감정 : 인사,인생
연령 : 30대
단어 : 선물,행운
말투 : 소설 문체


올 한 해, 여러분의 인생에 어떤 일들이 일어났었나요? 이번 크리스마스에는 산타클로스의 선물처럼 행운이 가득하길 기원합니다.

[예시 5]
시즌 : 연말
감정 : 인사,인생
연령 : 30대
단어 : 고생,행복
말투 : 소설 문체

올해 여러 가지 일들로 고생 많았지만 그만큼 얻은 것도 많았던 한 해였습니다. 다가올 새해에는 더 큰 행복과 축복이 가득하길 기원합니다.

[예시 6]
시즌 : 연말
감정 : 인사,희망
연령 : 40대
단어 : 복권,고생
말투 : 사극 말투

올 한해도 고생 많았소. 아직 우린 긁지 않은 복권이니 내년에도 새로운 희망을 품어 봄세!

[예시 7]
시즌 : 신년
감정 : 인사,공부
연령 : 10대
단어 : 대학,나비
말투 : 경상도 사투리

마! 새해 복 많이 받아라 ᄏᄏ, 수능 치느라 고생했제?, 대학가선 나비처럼 훨훨 날아래이

[예시 8]
시즌 : 신년
감정 : 사랑,인생
연령 : 20대
단어 : 연인,고백
말투 : 존댓말
